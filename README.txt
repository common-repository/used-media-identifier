== Description ==

Used Media Identifier is built to find and eliminate the risks of over-using same images for your posts from your WordPress media library.

An advantage to e-commerce business owners and designers- this plugin will automatically highlight the images that are identical from your existing gallery and file names via WordPress admin panel.

If you have a bulk of images and you are not able to find the used images which is used in different area in the website then this plugin will helps you to find the used images from media library. 



== Installation ==

This section describes how to install the plugin and get it working.

1. Download ZIP file from your WordPress website.
2. Log into your website administrator panel.
3. Go to "Add New" option from the plugin's dashboard and click on "Upload Plugin" then, click on "Install Now".
4. Ensure your image upload folder is writable and then click on "Activate Plugin".
6. You will see media section in wp-admin.
